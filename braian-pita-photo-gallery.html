Javascript and CSS Photo Galley

This photo gallery eases the visualization of images through a user friendly interface. This allows the quick showcase of pictures, which is also done in a visually pleasing way. This project also allowed me to understand more the use of CSS and Javascript, as well as how to work with code made by other people. I used an external library for the implementation of the gallery, which I edited to suit my needs.
