Current Absolute Path


This lab assignment presents a javascript code that will print the absolute path for the current web page. If this code is implemented in any HTML document, it will display the current path for that document. This path will be displayed on an element with the id "path". The lab helped me understand concepts of javascript programming like the use of the "document" object. Using properties and methods from the document object can allow you a better control over the website and managing it through javascript code.
