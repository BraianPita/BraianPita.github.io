Current Time


This lab assignment presents the use of a javascript code that, when executed, will print the current day of the week and time on the page. The purpose of this page is to be more appealing to the people visiting your website by adding more functionality to it. However, the creation of this page goes beyond just functionality, since it also taught me new helpful skills that can be used for future pages. The lab helped me understand the use of javascript and how to properly include javascript code on my HTML5 code. This allows endless possibilities for websites that are more functional and enjoyable to use.
